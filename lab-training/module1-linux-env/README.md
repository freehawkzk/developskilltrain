## 学习目标

完成本模块后，学生应能够：
- 在Linux终端下完成基本的文件操作和进程管理
- 通过SSH远程连接服务器
- 使用tmux让程序在后台运行
- 使用Conda管理Python环境
- 成功运行 `import torch` 并调用GPU
- 使用Jupyter Lab进行交互式开发
- 通过SSH端口转发在本地浏览器中使用服务器上的Jupyter
