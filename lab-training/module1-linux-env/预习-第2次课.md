## 你应该知道的概念

### 1. 虚拟环境
- 为什么需要？不同项目需要不同版本的库，混在一起会冲突
- Conda是最常用的Python环境管理工具
- 类比：每个项目一个独立的工具箱，互不干扰

### 2. CUDA
- NVIDIA GPU的编程接口
- PyTorch需要CUDA才能使用GPU
- CUDA版本必须和PyTorch版本匹配

### 3. 几个关键术语

| 术语 | 含义 |
|------|------|
| conda | 环境和包管理工具 |
| pip | Python包管理工具 |
| CUDA | NVIDIA GPU加速库 |
| Jupyter Lab | 交互式编程环境 |
| Docker | 容器化工具，打包整个环境 |

## 课前安装（强烈建议）

如果你有自己的笔记本，请**课前**完成以下安装：

### 步骤1：安装Anaconda或Miniconda

- Miniconda（推荐，更轻量）：https://docs.conda.io/en/latest/miniconda.html
- Anaconda（功能更全，体积更大）：https://www.anaconda.com/download

### 步骤2：创建环境

```bash
conda create -n lab-training python=3.10
conda activate lab-training
```

### 步骤3：安装PyTorch

先确认你的CUDA版本（如果有NVIDIA显卡）：
- Windows: 打开命令行，运行 `nvidia-smi`
- 如果没有NVIDIA显卡，安装CPU版本

然后到 https://pytorch.org/get-started/locally/ 选择对应的安装命令。

### 步骤4：安装其他依赖

```bash
conda install numpy scipy matplotlib scikit-learn jupyterlab tqdm
conda install -c conda-forge librosa soundfile
pip install pesq pystoi
```

## 环境检查清单

上课前请确认以下各项：

- [ ] `conda --version` 能正常运行
