# Vim速查表

> 本速查表只包含培训中需要用到的最基本操作。Vim功能极其强大，但初学只需要这些。

## 模式切换

| 按键 | 功能 |
|------|------|
| `i` | 进入插入模式（在光标前插入） |
| `a` | 进入插入模式（在光标后插入） |
| `o` | 进入插入模式（在当前行下方新开一行） |
| `Esc` | 退出插入模式，回到普通模式 |

## 保存与退出

| 命令 | 功能 |
|------|------|
| `:w` | 保存 |
| `:q` | 退出（未修改时） |
| `:wq` 或 `ZZ` | 保存并退出 |
| `:q!` | 不保存强制退出 |

## 移动光标（普通模式下）

| 按键 | 功能 |
|------|------|
| `h` / `j` / `k` / `l` | 左/下/上/右 |
| `w` | 跳到下一个词的开头 |
| `b` | 跳到上一个词的开头 |
| `0` | 跳到行首 |
| `$` | 跳到行尾 |
| `gg` | 跳到文件开头 |
| `G` | 跳到文件末尾 |
| `:行号` | 跳到指定行 |
| `ctrl+d` | 向下翻半页 |
| `ctrl+u` | 向上翻半页 |

## 编辑操作（普通模式下）

| 按键 | 功能 |
|------|------|
| `x` | 删除光标处字符 |
| `dd` | 删除（剪切）整行 |
| `yy` | 复制整行 |
| `p` | 在光标后粘贴 |
| `P` | 在光标前粘贴 |
| `u` | 撤销 |
| `ctrl+r` | 重做 |
| `.` | 重复上一次操作 |

## 搜索与替换

| 命令 | 功能 |
|------|------|
| `/关键词` | 向下搜索 |
| `?关键词` | 向上搜索 |
| `n` | 跳到下一个匹配 |
| `N` | 跳到上一个匹配 |
| `:%s/旧/新/g` | 全文替换 |
| `:%s/旧/新/gc` | 全文替换（逐个确认） |

## 常见场景

### 修改配置文件
```bash
vim config.yaml
# 按 i 进入插入模式
# 修改内容
# 按 Esc 退出插入模式
# 输入 :wq 保存退出
```

### 查看训练日志
```bash
vim train.log
# 用 G 跳到末尾查看最新日志
# 用 /error 搜索错误信息
# 用 :q 退出（不修改，直接退出）
```

### 快速修改一行
```bash
# 用 :行号 跳到目标行
# 用 0 跳到行首
# 按 cw 修改一个词
# 按 Esc 然后输入 :wq
```

## 提示

- 如果不小心进入奇怪的状态，多按几次 `Esc` 回到普通模式
- 不确定当前在什么模式？按 `Esc` 就对了
- Vim不是必须的——VS Code Remote是更友好的选择，Vim只用来做简单修改
