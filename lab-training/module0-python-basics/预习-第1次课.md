# 预习文档：模块0 第1次课 —— Python基础

> 请在上课前阅读本文档，了解本次课的核心概念和关键术语。

## 你应该知道的概念

### 1. Python是什么？
- 一种编程语言，语法简洁，适合快速开发
- 在深度学习和科学计算中广泛使用
- 你之前可能用它画过图或做过计算——这次课的目标是从"当计算器用"升级到"写程序"

### 2. Jupyter Notebook是什么？
- 一个交互式编程环境，代码和文字混排
- 你在浏览器中打开它，一段一段地运行代码
- 适合探索性数据分析和教学演示

### 3. 几个关键术语

| 术语 | 含义 | 你可以类比为 |
|------|------|-------------|
| 变量 | 存储数据的容器 | 物理中的符号（如 f 代表频率） |
| 函数 | 一段可复用的代码 | 数学中的函数 f(x) |
| 列表 | 有序的数据集合 | 一个有序的数组 |
| 字典 | 键值对的集合 | 一张查找表 |
| 数组 | 同类型数据的有序集合（NumPy） | 向量或矩阵 |

### 4. NumPy是什么？
- Python的数值计算库，提供数组和矩阵运算
- 类似MATLAB的矩阵运算能力
- 本课中用NumPy来生成和操作音频信号

## 思考题

1. 你之前用Python做过什么？遇到了哪些困难？
2. 如果你要描述一段声音的基本参数，你会用哪些量？（提示：频率、振幅、时长...）
3. 如果一段代码需要在3个地方重复使用，你有什么办法避免复制粘贴？
