# 模块0：Python编程基础

## 课程安排

| 课次 | 主题 | 课时 |
|------|------|------|
| 第1次 | Python基础——从计算器到程序 | 2.5h |
| 第2次 | Python进阶——面向对象与代码组织 | 2.5h |
| 第3次 | 调试、版本控制与编程习惯 | 2.5h |

## 编程能力目标

完成本模块后，学生应能够：
- 用Python写基本的数据处理脚本（函数、循环、条件判断）
- 理解类(class)的概念，能读懂别人写的类，能写简单的类
- 使用基本调试方法定位代码中的bug
- 使用Git进行基本的版本管理（init, add, commit, push, pull）
- 养成基本的编程习惯（命名规范、注释、代码格式化）

## 编程练习设计模式

本模块采用**填空式**练习——给出代码框架，学生填写关键部分。

## 数据与素材

所有练习均使用音频信号作为素材（正弦波生成、wav文件读取等），与后续模块自然衔接。

## 课前准备

- 确认Python 3.10+已安装
- 确认Jupyter Lab可以正常启动
- 确认numpy, matplotlib, scipy已安装
