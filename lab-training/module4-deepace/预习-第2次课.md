## 你应该知道的概念

### PyTorch关键API

| API | 用途 | 在DeepACE中的对应 |
|-----|------|------------------|
| nn.Conv1d | 一维卷积 | Encoder |
| nn.ConvTranspose1d | 一维转置卷积 | Decoder |
| nn.GroupNorm | 分组归一化 | ConvBlock中的归一化 |
| torch.fft / Conv1d | 频域/时域特征提取 | Encoder替代了FFT |
| nn.MSELoss | 均方误差损失 | 训练损失函数 |

### DeepACE代码结构

| 文件 | 功能 |
|------|------|
| config.yaml | 所有超参数配置 |
| model.py | 主模型类 DeepACE |
| netblocks.py | 子模块：Rectifier, ConvBlock, MaskGenerator, SELayer等 |
| dataset.py | 数据集加载（WAV + .mat） |
| train.py | 训练入口 |
| test.py | 测试入口 |
| losses.py | 多种损失函数 |

### 数据格式

| 数据 | 格式 | 形状 |
|------|------|------|
| 输入(mixture) | WAV, 16kHz单声道 | (1, 64000) — 4秒 |
| 目标(target) | .mat文件, 含'lgf'变量 | (250, 22) — 250帧×22通道 |
| 模型输出 | PyTorch tensor | (batch, 22, 250) |

## 核心概念：数据流

```
WAV波形 (64000样本)
    |
    v  Encoder: Conv1d(kernel=32, stride=16)
特征 (64通道, ~4000帧)
    |
    v  Rectifier
激活后特征 (64, ~4000)
    |
    v  MaskGenerator
掩码 (22, ~4000)
    |
    v  mask * features
掩码后特征 (64, ~4000)
    |
    v  Decoder: ConvTranspose1d
电极图 (22, ~250)
    |
    v  ChannelRebalancer + Hardtanh
最终输出 (22, ~250), 值在[1e-6, 1.0]
```
