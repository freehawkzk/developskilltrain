# 模块2：深度学习入门

## 课程安排

| 课次 | 主题 | 课时 |
|------|------|------|
| 第1次 | 从感知机到神经网络 | 2.5h |
| 第2次 | 卷积神经网络与声音的"图像化" | 2.5h |
| 第3次 | 训练技巧与实验方法论 | 2.5h |

## 学习目标

完成本模块后，学生应能够：
- 理解"学习"就是"调参数"的直觉
- 理解反向传播的直觉（不要求推导）
- 理解语谱图：音频可以变成"图像"来处理
- 用PyTorch搭建并训练CNN
- 知道如何评估模型、调参、做实验记录
- 理解过拟合/欠拟合及其应对方法

## 编程练习设计模式

本模块采用**修改式**练习——在已有代码上修改功能。部分环节仍保留填空式，但逐步过渡到需要学生自己编写代码的部分。

## 前置条件

- 模块0和模块1的全部内容
- PyTorch环境已配置完成
- Jupyter Lab可以正常使用

## 课前准备

- 确认所有学生能 `import torch` 且 `torch.cuda.is_available() == True`
