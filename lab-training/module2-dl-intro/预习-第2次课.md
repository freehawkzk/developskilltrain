## 你应该知道的概念

| 术语 | 含义 |
|------|------|
| MLP | 多层感知机 = 多层线性变换 + 激活函数 |
| 激活函数 | 引入非线性（ReLU, Sigmoid, Tanh） |
| ReLU | f(x) = max(0, x) ≈ 半波整流 |
| 反向传播 | 自动计算梯度的算法 |
| nn.Module | PyTorch中所有神经网络层的基类 |
| nn.Linear | 全连接层（线性变换） |
| nn.Sequential | 按顺序组合多个层 |
