## 你应该知道的概念

| 术语 | 含义 | 说明 |
|------|------|------|
| 采样率 | 每秒采样的次数 | 语音常用16kHz，音乐用44.1kHz |
| FFT | 快速傅里叶变换 | 将时域信号转换为频域 |
| 频谱 | 各频率成分的强度 | FFT的输出 |
| 语谱图(spectrogram) | 频率成分随时间的变化 | 横轴时间、纵轴频率、颜色=能量 |
| 梅尔刻度(Mel scale) | 模拟人耳频率感知 | 低频高分辨率、高频低分辨率 |
| MFCC | 梅尔频率倒谱系数 | 传统语音识别常用特征 |
| 预加重 | 提升高频成分 | y[n] = x[n] - 0.97·x[n-1] |
| 分帧加窗 | 将信号切成短段并加窗 | 帧长25ms，帧移10ms |

## 核心概念：特征提取链

```
原始波形 → 预加重 → 分帧加窗 → FFT → 梅尔滤波 → 对数 → MFCC
```

本节课会逐步走完这条链路，并在每一步可视化中间结果。
