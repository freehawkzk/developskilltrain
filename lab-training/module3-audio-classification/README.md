# 模块3：面向声音的分类模型

## 课程安排

| 课次 | 主题 | 课时 |
|------|------|------|
| 第1次 | 音频特征工程与经典方法 | 2.5h |
| 第2次 | 音频分类模型搭建 | 2.5h |
| 第3次 | CI相关音频分类实战 | 2.5h |

## 学习目标

完成本模块后，学生应能够：
- 理解音频特征的完整转换流程（波形→频谱→语谱图→MFCC）
- 理解梅尔刻度与CI电极映射的关系
- 用PyTorch搭建完整的音频分类pipeline（数据加载→特征提取→模型→训练→评估）
- 搭建CRNN音频分类器
- 将通用方法应用于CI相关任务（VAD、可懂度分类）

## 编程练习设计模式

本模块是"从修改式到组合式"的过渡。要求学生自己搭建完整pipeline，不再给填空式模板。

## 数据集

- ESC-50：环境声分类（50类，2000条）
- Speech Commands：语音命令识别（35类）
- 自制语音/噪声二分类数据集
