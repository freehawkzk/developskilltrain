## 学习目标

完成本模块后，学生应能够：
- 理解音频特征的完整转换流程（波形→频谱→语谱图→MFCC）
- 理解梅尔刻度与CI电极映射的关系
- 用PyTorch搭建完整的音频分类pipeline（数据加载→特征提取→模型→训练→评估）
- 搭建CRNN音频分类器
- 将通用方法应用于CI相关任务（VAD、可懂度分类）

## 编程练习设计模式

本模块是"从修改式到组合式"的过渡：
- 第1次课（notebook 01）：**修改式**——在已有代码上改特征提取函数
- 第2次课（notebook 02）：**组合式**——自己搭建 Dataset+DataLoader+Model+训练循环
- 第3次课（notebook 03）：**组合式+独立设计**——独立设计数据处理流程

课堂上给出数据流图而非代码——告诉学生"数据经过哪些变换，每步输入输出什么shape"，让学生自己查API写代码。
