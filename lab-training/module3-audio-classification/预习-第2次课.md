## 你应该知道的概念

### PyTorch 数据加载

| 术语 | 含义 |
|------|------|
| Dataset | 定义如何加载单条数据（__getitem__） |
| DataLoader | 负责批量组合、打乱顺序 |
| collate_fn | 自定义如何将多条数据组合成batch |

### 模型架构

| 架构 | 特点 |
|------|------|
| CNN | 提取局部时频模式，忽略全局时序 |
| RNN/GRU | 建模时序依赖，理解"先后顺序" |
| CRNN | CNN + RNN 的组合，音频分类首选 |
| Attention | 聚焦关键帧，更灵活 |

### 评估指标

| 指标 | 含义 |
|------|------|
| 混淆矩阵 | 展示每个类别被分对/分错的详细情况 |
| Precision | 预测为某类中，真正是该类的比例 |
| Recall | 真正是某类中，被正确预测的比例 |
| F1 | Precision和Recall的调和平均 |

## 核心思想

CRNN = CNN找"模式" + RNN理解"模式的顺序"

```
语谱图 → CNN(局部模式) → GRU(时序) → FC(分类)
```

关键设计：CNN的池化**只在频率维度**下采样，保留时间维度给GRU处理。
