## 前置要求

本次课是代码实战课，需要完成以下准备：

### 1. DeepFilterNet论文回顾（约15分钟）

确保你已经理解上节课的核心概念：
- 两阶段架构：Enhancement（ERB掩码）+ Deep Filtering（时域滤波）
- ERB频带的含义和作用
- alpha参数的含义（控制DF阶段的混合比例）

如果论文阅读引导卡中的问题还有未完成的，请先补上。

### 2. 代码预浏览（约30分钟）

打开 `DeepFilterNet-main/DeepFilterNet/df/` 目录，快速浏览以下文件（不需要读懂每一行）：

| 文件 | 浏览重点 |
|------|---------|
| `config.py` | 找到 `DfParams` 类，注意默认参数值（SR、FFT_SIZE、NB_ERB等） |
| `deepfilternet.py` | 找到 `DfNet` 类的 `forward()` 方法，理解数据流 |
| `modules.py` | 浏览类名列表（Conv2dNormAct、GroupedGRU、Mask、DfOp等） |
| `enhance.py` | 找到 `init_df()` 和 `enhance()` 函数签名 |

浏览策略：先看类和函数的名字、参数列表、docstring，暂时跳过具体实现。

### 3. 环境安装（约15分钟）

请课前完成DeepFilterNet的安装（两种方式选一种）：

**方式A：pip安装（推荐）**
```bash
pip install deep-filter
```

**方式B：从源码安装（需要Rust工具链）**
```bash
cd DeepFilterNet-main/DeepFilterNet/
pip install -e .
```

验证安装：
```bash
python -c "from df.enhance import init_df; print('OK')"
```

如果安装失败也无需担心，课堂上有备选方案。

### 4. 预训练模型解压（约5分钟）

```bash
cd DeepFilterNet-main/models/
unzip DeepFilterNet3.zip
```

确认解压后能看到 `DeepFilterNet3/` 目录。
